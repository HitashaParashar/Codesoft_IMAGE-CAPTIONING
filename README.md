# Image Captioning AI

## Project Overview
An image captioning system using pretrained ViT + GPT-2 model.

## Tech Stack
- Python
- PyTorch
- HuggingFace Transformers
- Flickr8k Dataset
- Google Colab

## How it works
1. ViT extracts image features
2. GPT-2 generates captions word by word

## Results
- a man on a skateboard jumping over a rail
- a basketball player is jumping up to hit a ball
- a white bird with a long beak is standing on a body of water
